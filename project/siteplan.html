Personal Portfolio Site Plan

Site Name

Name: Sam Daramroei Portfolio

Reason: The name reflects my personal brand and highlights my expertise in design, web development, and creative fields.

Site Purpose

The purpose of this site is to showcase my abilities, personality, and portfolio to potential clients. The site will include sections on my skills, past works, personality, education, and services offered, making it a central hub for professional opportunities.

Scenarios

  • Scenario 1: A client looking for a web designer visits my portfolio to see if my style matches their project needs.
  • Scenario 2: A business owner interested in hiring a graphic designer explores my past projects for inspiration and credibility.

Color Schema

  • Primary Color: #003366 - Used for the header, footer, and headings.
  • Accent Color: #F4C430 - Used for call-to-action buttons and highlights.
  • Background Color: #f8f4e3 - Provides a soft, neutral background.

Typography

  • Font 1: Montserrat - Used for headings to ensure a clean and professional look.
  • Font 2: Poppins - Used for body text for readability and a modern feel.

Wireframe

Wireframe sketches will include:

  • Mobile view: A single-column layout with a collapsible menu.
  • Desktop view: A two-column layout with navigation on the side and content in the main area.
